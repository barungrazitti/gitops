# Quick Plan: Fix binary file diff detection generates incorrect commit message

**Quick ID:** 260410-h09
**Date:** 2026-04-10

## Task

Fix binary file diff detection generates incorrect commit message

## Problem

When adding/removing PDF or other binary files, the AI generates incorrect commit messages like "chore(pages-pdf): remove suva-analytics-report.pdf" instead of proper messages like "chore: add suva-analytics-report.pdf"

Issue: Binary files have no +/- changes in the diff (git shows them as "Binary files a/x b/y differ"), so the AI has no content to analyze.

## Solution

1. Detect binary files in efficient-prompt-builder.js by checking for lack of +/- lines
2. Add special handling: generate factual message based on file path, not AI analysis

## Tasks
